FICTOAN is an intuitive, simple SCSS framework that also moonlights as an UI kit.

Give it a shot— download or fork from Github.

It’s intuitive.

FICTOAN uses straight-forward and intuitive class names like .horizontal-center-this, .no-padding-left, .center-on-mobile and so on. Simple to use, and easy to recollect.

It’s modular.

Styling elements is like a jigsaw puzzle— take a .button, simply add .bg-red, .shape-circular and .no-shadow to get a red, round button without a shadow. It’s like playing with Lego.

It’s responsive.

Change layouts across devices— .one-fourth, .one-third-on-tab-ls, .six-eights-on-mobile and so on. Or use .retain-layout-on-mobile to turn off responsiveness completely.

It’s customisable.

It’s also got a nice little variables file to tweak. Now you can have different styling rules for each of your projects, without having to rewrite each time.

It’s ready to go.

It comes with a set of pre-defined elements that you can simply drop into your project. Cards, sidebar menus, modal windows and whatnot.

It’s open source.

Just head to the Github page and fork at will. Not interested in writing code? Just add your feedback, and it might just make it to the next feature release!
