FICTOAN

(F*ck, I Couldn't Think Of A Name)

FICTOAN is an intuitive, minimalist, responsive HTML+CSS boilerplate.

Most sites be like, “For devs, by devs”. So, this is for designers, by a designer.

FICTOAN uses straightforward, self-explanatory and intuitive class names. No cols, spans, MDs, LGs and such.

This is mainly for newbies just starting to learn code(moi), but are put off by complex systems like Bootstrap or Foundation.

Right now it’s got a grid, buttons, typography, icons, colours and code. More modules coming soon.

Installing

You can DOWNLOAD the source files directly and get right to work— there’s a template HTML file where you just have fill content.

Or, if you’re the slightly more nerdy type, there’s also a GitHub repo you can clone, fork, edit, have a good laugh, whatever.

The naming system

FICTOAN uses names like .horizontal-centre-this, .no-margin-top, .shape-circular and .only-on-mobiles. Simple and easy to recollect.

The styling is also modular— throw button, .large, .no-bg and .shape-rounded together to get a large, borderless button that has rounded edges.

The Grid

A digital frontier...I tried to picture clusters of infor— oh. Right. Sorry.
The grid is made up of full-width .row classes that can contain one or many .portion elements. These portions is where you put your content. All the rows put together form one big box called .all-content.

The .portion elements are, well, portions of the row, and come in varying widths, from .whole —which takes up the entire row, all the way to .one-eighth, which takes up 12.5% of the width. Different combinations are possible, as long as the sum of their widths is 100%. Well, they can be less, but don’t leave them hangin’, they don’t like that.

Typography

FICTOAN uses Cormorant Garamond for a serif typeface, Work Sans for sans, and Fira Mono for monospace. They’re all from Google Fonts, and thus, free.

Headings are set in Cormorant Garamond, and sizes vary between 4em for h1 and 1em for h6. Want some more flexibility? Text also comes in—

huge, large, big, small and tiny sizes. Simply use .text-huge, .text-big and so on.

Now, the usual bold, italic and link styling. Throw serifs into the mix, and you have Cormorant Garamond that’s bold, italic and also a link. You can even combine these with the above text size classes, but handle with care.

But wait, there’s more! You can also style inline text as serif, sans-serif or monospace using .text-serif, .text-sans and .text-mono. Neat, huh?

Icons

There’s also a sweet set of 16 custom-made, pixel-perfect SVG icons that are completely free to use. They’re all drawn on a 24px grid, and will scale nicely at its multiples. You can even scale them with CSS, just use .icon-24 or .icon-48 to resize as you need. More icons will be added regularly.

Colour

FICTOAN comes with a bright and vivid scheme of eight colours. You can apply them to pretty much any element as a background colour, or to text too. Plus, there’s also a tint and a shade— 5% black and 85% black, which you can customise as necessary.

For applying colours to text, just use .text- as a prefix, instead of .bg-. And, if you use the icons as an inline element instead of an image, you can even colour them with CSS classes! All you need is the .icon- prefix. How cool is that?

Buttons

Buttons are a cinch with FICTOAN. First add the .button class, and then any combination of the styling such as .shape-circular, .button-large, .no-shadow and so on.

Toggles & Radio buttons

Toggle switches are just restyled checkboxes. The radio buttons are also styled similarly, but the code is separate, which means you can customise either one individually. Just use a p tag after the toggle for the label.
